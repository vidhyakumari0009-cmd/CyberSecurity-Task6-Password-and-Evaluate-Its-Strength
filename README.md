#Password Strength Evaluation

## Objective
Understand strong password creation and evaluate password security.

## Tool Used
Password Tester

## Steps
1. Created multiple passwords
2. Tested strength using online checker
3. Compared scores
4. Learned password security practices

## Concepts Covered
- Password Strength
- Brute Force Attack
- Dictionary Attack
- Authentication

## Outcome
Learned how password complexity improves security.
